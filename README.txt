# Hasseef Leadership Demo (Bilingual)
- `index.html` — Executive-grade, bilingual (AR/EN), fully offline.
- `styles/theme.css` — Clean government-grade theme.
- `data/KPIs.json`, `data/Accounts.json` — Data for KPIs and role accounts.
- `assets/logos/` — Logos (Vision 2030 / Hasseef / Talbiya).

## How to run
Open `index.html` directly in a modern browser. No server required.
